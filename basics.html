HTML Basic Examples

Don't worry if these examples use tags you have not learned yet. You will learn them in the next chapters.

HTML Documents

All HTML documents must start with the declaration <!DOCTYPE html>. The HTML document begins with <html> and ends with </html>. The visible page content is between <body> and </body>.

Example


<!DOCTYPE html>
<html>

<body>

<h1>My First Heading</h1>

<p>My First Paragraph.</p>

</body>

</html>

HTML Paragraphs

HTML paragraphs are defined with the <p> tag.

Example


<p>This is a paragraph.</p>

<p>This is another paragraph.</p>

HTML Links

HTML links are defined with the <a> tag.

Example


<a href="https://www.google.com/">
This is a link
</a>

The link address is specified in the href attribute. Attributes provide additional information about HTML elements.

HTML Images

HTML images are defined with the <img> tag. The image source (src), alternative text (alt), width and height are provided as attributes.

Example


<img src="yourimage.jpg"
alt="your image text"
width="104"
height="142">
